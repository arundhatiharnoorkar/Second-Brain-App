# Second Brain - Personal Knowledge Management System

A full-stack note management application to create, organize, search and manage personal notes.

## Features

- User authentication with JWT
- Create, update, delete notes
- Pin important notes
- Category and tag based organization
- Search notes
- Dark mode UI

## Tech Stack

Frontend:
- React
- TypeScript
- Tailwind CSS

Backend:
- Node.js
- Express.js
- Prisma
- PostgreSQL

## Run Locally

Frontend:
npm install
npm run dev

Backend:
npm install
npm run dev
